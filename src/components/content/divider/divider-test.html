Component Features

  • ✅ Self-contained (token-led from cancara-tokens.css)
  • ✅ Two width variants: Default (with padding) and Fullwidth
  • ✅ Two type variants: Default (solid) and Dash (dashed)
  • ✅ Fully responsive (100% width within container)
  • ✅ Dark mode support
  • ✅ Uses design tokens: --divider-bg, --divider-height, --divider-margin
  • ✅ Accessible (semantic structure)
  • ✅ No hardcoded widths (unlike original 343px/375px)
