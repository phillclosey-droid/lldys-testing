Component Features

  • ✅ Two layouts: Default (label above), Inline (label beside)
  • ✅ Four states: Default, Active, Error, Disabled
  • ✅ Self-contained (token-led from cancara-tokens.css)
  • ✅ Clear button (auto-hides when empty)
  • ✅ Error states with icon and messages
  • ✅ Hint messages for guidance
  • ✅ Dark mode support
  • ✅ Accessible (proper labels, ARIA attributes)
  • ✅ Keyboard navigation
