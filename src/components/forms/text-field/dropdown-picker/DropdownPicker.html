# Dropdown Picker Component A dropdown selector component with label, value text, and chevron icon. Available in default (vertical) and inline (horizontal) layouts. ## Files - `DropdownPicker.vanilla.css` - Self-contained CSS with token-led styling - `DropdownPicker.vanilla.js` - Interactive functionality - `DropdownPicker-demo.html` - Demo page showing all variants ## Component Structure ### Default Layout (Vertical) ```html
Select one option
``` ### Inline Layout (Horizontal) ```html
Label : Select one option
``` ## CSS Classes ### Main Container - `.dropdown-picker` - Main container - `.dropdown-picker.layout-inline` - Inline/horizontal layout - `.dropdown-picker.is-disabled` - Disabled state ### Label - `.dropdown-picker-label` - Label for default layout (type-style-6, 15px) - `.dropdown-picker-label-inline` - Label inside input for inline layout - `.dropdown-picker-colon` - Colon separator in inline layout ### Input Components - `.dropdown-picker-input-wrapper` - Wrapper for input - `.dropdown-picker-input` - Input container - `.dropdown-picker-input.is-active` - Active/focused state (2px black border) - `.dropdown-picker-content` - Content wrapper for inline layout - `.dropdown-picker-value` - Value display text ### Icon - `.dropdown-picker-icon` - Icon container (16px × 16px) - `.dropdown-picker-icon-button` - Icon button for inline layout ## JavaScript Events The component dispatches a custom `dropdown-picker-click` event when clicked: ```javascript element.addEventListener('dropdown-picker-click', function(e) { console.log(e.detail.isActive); // true if active state is toggled on }); ``` ## Design Tokens Used ### Typography - `--type-style-6-family` - Label and value font (GT Ultra) - `--type-style-6-size` - Label and value size (15px) - `--type-style-6-weight` - Font weight - `--type-style-6-line-height` - Line height (20px) ### Colors - `--text-default` - Label and value text (changes in dark mode) - `--text-input-disabled` - Disabled text color (gray-500) - `--bg-input-default` - Input background (white) - `--bg-input-disabled` - Disabled background (gray-100) - `--border-default` - Border color (gray-300 light, gray-600 dark) - `--color-black` - Active border (light mode) - `--color-white` - Active border (dark mode) ### Spacing - `--spacing-08` - Gap between elements (8px) - `--spacing-12` - Gap between input and icon button (inline layout, 12px) - `--spacing-16` - Container padding, input padding (16px) ### Borders - `--input-corner-radius` - Input border radius (12px) - `--input-border-width-default` - Default border (1px) - `--input-border-width-active` - Active border (2px) - `--input-min-height` - Minimum input height (56px) ## Dimensions - **Input padding**: 16px - **Border radius**: 12px - **Icon**: 16px × 16px - **Gap (inline layout)**: 12px between input and icon button - **Min height**: 56px ## States ### Default - White background - 1px border using `--border-default` - Black text - Chevron icon visible (inside for default, outside for inline) ### Active/Focused - 2px black border (white in dark mode) - Triggered by click ### Disabled - Gray-100 background - Gray-500 text - No chevron icon (default layout) or disabled icon button (inline layout) - Not clickable ## Behavior - Clicking input toggles active state - Clicking icon button (inline layout) toggles active state - Clicking outside removes active state - Disabled inputs cannot be clicked - Custom event dispatched on click ## Icon **Location:** `assets/icons/arrows/chevron-down.svg` - **Size:** 16px × 16px - **Format:** SVG - **Dark mode:** Icon inverts color automatically - **Position:** Inside input for default layout, outside as button for inline layout ## Usage Examples ### Basic Dropdown Picker ```html
