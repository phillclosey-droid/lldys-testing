Component Features

  • ✅ Self-contained (includes nested link-button and divider styles)
  • ✅ Token-led from cancara-tokens.css
  • ✅ Two directions: Horizontal and Vertical
  • ✅ Three button states: Default, Pressed/Hover, Disabled
  • ✅ Supports 1-n link buttons
  • ✅ Automatic separators (vertical line for horizontal, divider for vertical)
  • ✅ Rounded bottom corners (12px)
  • ✅ Dark mode support
  • ✅ Fully responsive (100% width)
  • ✅ Accessible (proper button semantics)
