Component Features

  • ✅ Self-contained layout component
  • ✅ Uses ActionButton as nested component
  • ✅ Two layouts: Vertical and Horizontal
  • ✅ Three variants: Default (all 3), Primary+Secondary, Primary+Tertiary
  • ✅ Token-led spacing (16px gap)
  • ✅ Horizontal layout reduces button padding (24px vs 48px)
  • ✅ Vertical layout has full-width buttons
  • ✅ Dark mode support via nested ActionButtons
