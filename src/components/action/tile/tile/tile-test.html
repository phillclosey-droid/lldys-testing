Component Features

  • ✅ Two tile orientations: horizontal, vertical
  • ✅ Two layout patterns: single (full-width), double (50% each)
  • ✅ Three presets: link-only, link-description, heading-description
  • ✅ Self-contained (includes Pictogram styles)
  • ✅ Token-based styling
  • ✅ Auto-initializes images via JavaScript
  • ✅ Dark mode support
  • ✅ Hover and active states
  • ✅ Multiple icon sizes: 12px, 18px, 24px
  • ✅ Multiple pictogram color variants
  • ✅ Flexible - supports custom grid layouts (triple, quad, etc.)
