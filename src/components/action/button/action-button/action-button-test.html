Component Features

  • ✅ Three types: Primary, Secondary, Tertiary
  • ✅ Three states: Default, Pressed, Disabled
  • ✅ Two modifiers: Brand Accent (green), On Color (inverted)
  • ✅ Self-contained (token-led from cancara-tokens.css)
  • ✅ Optional arrow icons (auto-initialized via JavaScript)
  • ✅ Dark mode support (primary buttons become green)
  • ✅ Hover and active states
  • ✅ Accessible (proper button semantics)
  • ✅ Text centered, icons positioned far right
