# Mileage Graph Component Token-led mileage visualization component with bar chart displaying annual mileage comparison. ## Files - `MileageGraph.vanilla.css` - Token-led styling - `MileageGraph.vanilla.js` - Optional interactive functionality - `mileage-graph-test.html` - Demo/test file ## Component Structure ```html
11,886 miles
Mileage for your 2025 MOT
12k
8k
4k
0
2024
2025
``` ## Features - **Token-led design**: All colors, spacing, and typography use design tokens - **Flexible data**: Bar heights can be adjusted via inline styles - **Grid system**: Dashed grid lines for value reference - **Rounded bars**: Top-rounded bars with centered dot indicators - **Responsive typography**: Baseline-aligned header text ## Design Tokens Used - Background: `--bg-panel-default` - Border radius: `--radius-default` - Typography: `--type-style-3-*`, `--type-style-6-*`, `--type-style-8-*` - Spacing: `--spacing-08`, `--spacing-12`, `--spacing-16`, `--spacing-24` - Colors: `--text-default`, `--text-subdued`, `--border-default`, `--datavis-purple-default` ## Customization Adjust bar heights by changing the `style="height: X%"` inline style on `.mileage-graph-bar` elements. Grid labels can be modified to match your data range.
