3-Tabs-Fixed Component

A three-tab navigation component with two layout variants (pill and underline styles). Self-contained, token-led, and fully interactive.

Overview

The 3-Tabs-Fixed component provides a simple three-tab navigation pattern with automatic dark mode support and two distinct visual styles.

Tab 01 Active
Tab 02 Active

JavaScript API

Auto-initialization

The component automatically initializes on page load. Simply include the JavaScript file:

<script src="ThreeTabsFixed.vanilla.js"></script>

Manual Initialization

// Initialize all 3-tabs-fixed components
window.ThreeTabsFixedComponent.init();

Events

Listen for tab changes:

const component = document.querySelector('.tabs-3-fixed');
component.addEventListener('tab-changed', function(e) {
  console.log('Tab index:', e.detail.tabIndex);
  console.log('Tab element:', e.detail.tabElement);
});

Dimensions

  • Container max-width: 375px
  • Default layout height: 48px (outer), 36px (tabs)
  • Default layout padding: 16px horizontal, 8px vertical
  • Alt 01 layout padding: 16px all sides
  • Active border height: 2px
  • Border radius: 12px (default layout corners)
