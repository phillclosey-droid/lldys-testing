Component Overview

Mobile bottom navigation bar with 5 tabs, active state indicator, and iOS home indicator. Fully token-based styling with light and dark theme support.

  • 5 navigation tabs with icons and labels
  • Active state with green top border indicator
  • Icons swap between normal and active states
  • iOS home indicator at bottom
  • Fully responsive width
  • Token-based styling for colors, spacing, typography
  • Light and dark theme support
  • Interactive - click tabs to switch active state
